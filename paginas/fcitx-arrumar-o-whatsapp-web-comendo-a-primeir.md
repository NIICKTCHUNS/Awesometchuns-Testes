# (Fcitx) Arrumar o WhatsApp Web comendo a primeira letra com acento no Firefox   
Instale os pacotes   
`fcitx5 fcitx5-configtool fcitx5-breeze fcitx5-qt`   
Adicione a seguinte linha no /etc/environment   
`GTK\_IM\_MODULE=fcitx5`
